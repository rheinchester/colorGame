# colorGame

A web application where individuals can display their work, for appreciation, validation and easy exposure. Photos, videos, research papers, you name it!!!.

- Easy registration
- Quick upload
- Clean interface

Development is still in progress, although almost done.......

# Tech involved
- Javascript 
- HTML & CSS
- Bootstrap
